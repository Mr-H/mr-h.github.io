# Startup Technology Business Website

Welcome to the Startup Technology Business website! This project is a static website designed to showcase our innovative technology solutions and services.

## Project Structure

- `src/`: Contains all source files for the website.
  - `css/`: Contains stylesheets for the website.
    - `style.css`: Main styles for layout, typography, and responsive design.
    - `normalize.css`: CSS reset for consistent styling across browsers.
  - `js/`: Contains JavaScript files for interactivity.
    - `main.js`: JavaScript code for handling dynamic content.
  - `pages/`: Contains HTML pages for the website.
    - `about.html`: About page with information about the startup.
    - `contact.html`: Contact page with a form or contact details.
    - `products.html`: Page showcasing the products offered.
    - `services.html`: Page outlining the services provided.
  - `components/`: Contains reusable HTML components.
    - `header.html`: Header structure including logo and navigation.
    - `footer.html`: Footer structure with copyright and links.
    - `navigation.html`: Navigation menu structure for easy access to pages.
- `docs/`: Contains documentation files.
  - `style-guide.md`: Style guide detailing design principles and component usage.
- `index.html`: Main entry point of the website.
- `.gitignore`: Specifies files to be ignored by Git.

## Getting Started

To get started with this project, clone the repository and open the `index.html` file in your web browser. You can also use a local server for better performance and to enable JavaScript functionality.
